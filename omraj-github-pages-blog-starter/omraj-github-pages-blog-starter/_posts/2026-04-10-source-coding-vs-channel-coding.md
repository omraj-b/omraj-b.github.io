This is a sample Markdown blog post.

You can write your posts in Markdown and place them inside the `_posts` folder.

## Format you should use

Each post filename should look like this:

`YYYY-MM-DD-title.md`

Example:

`2026-04-10-source-coding-vs-channel-coding.md`

## Front matter

Every post should start with front matter like this:

```yaml
---
layout: post
title: "Your Post Title"
date: 2026-04-10
tags: [tag1, tag2]
description: "Short summary"
---
```

After that, write normal Markdown.
