## How to add a new blog post
1. Create a new Markdown file in `_posts/`
2. Name it `YYYY-MM-DD-title.md`
3. Add Jekyll front matter at the top
4. Commit and push to GitHub
5. GitHub Pages will rebuild the site and the post will show up on `blog.html`

## Important
- This uses GitHub Pages' built-in Jekyll processing.
- Keep the repository public if you are on GitHub Free and want Pages enabled.
- If your repo is `username.github.io`, this will work as a user site.
